# Assets

This directory contains app assets (icons, images, splash screens).

## Required Assets

To complete the app setup, you need to add the following image files:

1. **icon.png** (1024x1024px) - App icon
2. **adaptive-icon.png** (1024x1024px) - Android adaptive icon
3. **splash.png** (1242x2436px) - iOS splash screen
4. **favicon.png** (48x48px) - Web favicon

You can generate these assets using tools like:
- Figma
- Adobe Illustrator
- Online icon generators

Or use Expo's asset generation service when building the app.
